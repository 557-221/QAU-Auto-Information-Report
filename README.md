# QAU-Auto-Information-Report
## 简介
这个Python脚本用于实现青岛农业大学每日信息上报的自动化  
  
* 这个项目的初衷是用于自己练习Git与GitHub的使用，对于Python仅是入门水平，代码很粗糙，许多实现应该有更为优秀的方法，但限于自身水平无法实现。  
如果您对于代码有更好的建议可以向我提供。

## 使用说明
* 配置    
配置前的准备工作：  
此脚本依赖于Python3以及Requests库  
Python3请自行安装  
Requests    ` pip install requests `
* 使用  
打开main.py  
对其中的User内的内容进行填写  
然后！Run！
